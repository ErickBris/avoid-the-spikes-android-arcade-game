This is a template for an Android game. The aim of the player is to help the bird move up and avoid the spikes/obstacles. When the screen is tapped, the bird changes direction. Thanks for buying my code from CodeCanyon!

A) Step By Step Customisation - top

  1. Import into Android Studio

    Open Android Studio, click on 'Open an Existing Android Studio Project', locate the contents of the 'CODE' file and click 'ok'.

  2. Test it!

    Just hit the play button to test that everything is ok.

  3. Replace any PlaceHolder Images and Sound

    In Android Studio you can change any images you wish. At least make sure to change the icon. To replace the images just copy and paste your new images in res/drawable. For the character, 4 different images are provided in the graphics folder. Just replace drawable/character.png with any of the 4 images you wish. The Sound fx/music are just placeholders. Feel free to replace them.

  4. Rename the App Package id

    First rename the package id by right clicking on 'com.neurondigital' and selecting 'refactor' and 'rename' as shown below. Do the same for 'avoidthespikes'. The package name should be in this format: com.yourcompany.AppName

  5. Open avoidthespikes/build.gradle and put in the same package id as before instead of the 'com.neurondigital.avoidthespikes'. This is shown below.

  6. Create an interstitial Ad on AdMob and paste the AD unit Id in the Strings.xml file (optional).

    The Ad displays once before gameover. Make sure to add this to start making money from your App. The display rate can be changed from strings.xml from the ad_shows_every_X_gameovers parameter.

  7. Create an Banner Ad on AdMob and paste the AD unit Id in the Strings.xml file (otional).

    The Ad displays in the bottom of the screen. Ideally it's either the banner or the interstitial ad, so you may wish to leave one of the IDs in strings.xml empty.

  8. Google Play Services (optional).

    The game template supports a top score leaderboard and 5 Achievements. To enable this feature, create the necessary leaderboard/achievements on Google Play. Then just copy the Generated App id and paste it in the Strings.xml file.

    Google Play Services Achievements.

    The game template can unlock google play Achievements when the player collects a certain amount of coins. To enable this just create 5 achievements on Google Play Store and put the generated App id (same as leaderboard) and the 5 achievement ids strings.xml:

  9. Google Analytics (optional).

    Create a new google Analytics property and you'll be provided with a tracking id which should have this format: "UA-xxxxxxxx-xx". Put this tracking id in strings.xml in the 'trackingid' field.

  10. Export your App as a .APK for Google Play

    Go to Build| Generate Signed APK... and export the .APK file.

  1. TODO Tags

    In the code you will find TODO tags. Make sure to check them out. They indicate the places that you usually need to change something, although in this app most of the things that would be changed are all in the strings.xml file.

  2. Change Colors

    Open Strings.xml. In the code search for the following snippet. Change the color codes, to change the respective colors.

  3. Change Game Variables

B) Editing Images - top

Most of the images are included in the design.ai file. So feel free to edit them as you like. Remember to keep the same image resolution before replaceing it with the one in Android Studio.
